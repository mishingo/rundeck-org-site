Debian/Ubuntu Install

{{site.rd_deb_name}}

Execute:

Yum RPM Install

Execute:

yum install java-1.8.0
rpm -Uvh http://repo.rundeck.org/latest.rpm
yum install rundeck
service rundeckd start

Or fetch from bintray:

https://bintray.com/rundeck/rundeck-rpm
